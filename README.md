# 🚀 Team Productivity Suite

A modern, full-stack web application designed to help teams collaborate, track tasks, and manage project workflows efficiently.

---

## ✨ Features

* **Full-Stack Architecture:** Separate Frontend UI and Backend API server.
* **Task Tracking:** Clean dashboard to manage daily project updates.
* **User Authentication:** Secure Login and Sign-In interface.

---

## 🛠️ Project Structure & Tech Stack

### 💻 Frontend
* **Tech:** HTML5, CSS3, JavaScript
* **What it does:** Beautiful gradient login page, responsive forms, and user dashboards.

### ⚙️ Backend
* **Tech:** Node.js / Express (or your backend language)
* **What it does:** Handles API requests, user routing, and data logic.
